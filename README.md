# passgen-cli - rust-based cli password generator

## Usage

```bash
Usage: passgen-cli [OPTIONS]

Options:
  -L, --length <LENGTH>  [default: 16]
  -m, --mode <MODE>      [possible values: num, lower, upper, special, alpha, alnum, all]
      --pre <PRE>        
      --post <POST>      
  -n, --numeric          
  -u, --upper            
  -l, --lower            
  -s, --special          
  -c, --custom <CUSTOM>  
  -h, --help             Print help
  -V, --version          Print version

```
